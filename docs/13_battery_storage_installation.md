# SunGrid Cooperative — Battery Storage Installation Requirements

## Compatible Configurations
Battery storage systems must be installed alongside a compatible inverter configuration. Systems using micro-inverters require a battery unit with AC-coupling support; systems using a string inverter may use either AC- or DC-coupled battery configurations, per the inverter guidance in the Technical & Installation Guidance document.

## Fire and Safety Code
Battery units must be installed with a minimum 3-foot clearance from doors, windows, and combustible materials, per SunGrid's adopted fire code standard. Indoor installations require a dedicated ventilation assessment by the installer prior to energizing.

## Capacity Sizing
SunGrid recommends sizing battery capacity to cover 50-100% of a household's average daily consumption, based on the site assessment generation data. Oversized systems beyond 150% of average daily consumption are permitted but are not eligible for additional incentive consideration beyond the standard Battery Storage Add-On Incentive cap.

## Commissioning
Battery systems undergo a separate commissioning test from the solar array itself, verifying charge/discharge behavior and grid-disconnect (islanding) safety response before being cleared for full operation.
